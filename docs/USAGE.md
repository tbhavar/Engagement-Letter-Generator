# Usage Guide

## Generating Engagement Letters

### Step 1: Access the App
- Open the Streamlit Cloud link
- Enter your name and password

### Step 2: Generate Letter
1. Select service type (Audit, Tax, GST, etc.)
2. Enter client details (name, PAN, etc.)
3. Enter your firm details
4. Describe the service
5. Click "Generate Engagement Letter"

### Step 3: Review
- Check the preview
- Download as text if needed
- Make edits in Word if necessary

### Step 4: Save
- Click "☁️ Save to GitHub"
- Letter automatically saved to cloud
- Appears in "All Letters" immediately

## Viewing Past Letters

1. Go to "📚 All Letters" tab
2. See all approved letters with metadata
3. Click to view details
4. See who created it and when

## Troubleshooting

- **Can't log in?** Check password
- **Can't see past letters?** Refresh page or wait 30 seconds
- **Can't save to GitHub?** Check GitHub token in Streamlit Cloud secrets
